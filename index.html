Troido's games and experiments

Projects

  • Airships. A physics-based game about building and using airships
  • Elaborate. A generator for procedural islands.
  • Bubblevania. Competitive bubble wrap popping simulator with combos. Created as part of team Snappy Cobra for Global Game Jam 2025.
  • Lout. Use instruments to keep the culture barbarians at away from the drums in this musical tower defence game. Created as part of team HedgeLizards.
  • Tomb Raiser. Experimental turn-based strategy game. Created as part of team HedgeLizards.
  • Gobble. Shoot those pesky knights away in this cooperative top down shooter. Created as part of team HedgeLizards.
  • Dezl. Multiplayer game of exploration, crafting and farming. Can be played both in the terminal as in your browser.
  • Collision Cats. Bounce your friends into the water and be the Last Cat Rolling. Created as part of team HedgeLizards for the Global Game Jam 2024.
  • Major. 3D and multiplayer remake of minor. Created as part of team HedgeLizards.
  • Yeeti. Experimental game about a Yeti that yeets the hunters. Created as part of team HedgeLizards.
  • Uproot. Race your friends to be the first to the surface in the arcade potato game! Created as part of team HedgeLizards for he Global Game Jam 2023. Winner of "The complete package" award.
  • Ship Happens. Yer ship be shot in half! Use yer grappling hook to maneuver and smash the enemy fleet! Created as part of team HedgeLizards.
  • Minor. Experimental diggin/vehicle construction game. Created as part of team HedgeLizards.
  • Cathy's guarden. A cute farming game. Surely those bunnies are completely innocent... right? Created as part of team HedgeLizards. Winner of the Global Game Jam NL 2022.
  • Under your Wing. Escape the collapsed mine using both the miner and the canary to solve puzzles. Created as part of team HedgeLizards for the Global Game Jam 2021.
  • Wegindeweg. Move mountains to steer the road away from your grain fields. Created as part of team Snappy Cobra for the Global Game Jam 2020.
  • ToiletBoy. A roguelike for the GameBoy Advance about waking up in the night and having to go to the toilet. Created as part of team Snappy Cobra for the Global Game Jam 2019.
  • Orbox. My version of the orbox puzzle game.
  • Minesweeper. A minesweeper clone. I couldn't find a minsweeper version with an interface I liked elsewhere so I made this.
  • Achtung die Kurve. An html remake of Achtung die Kurve. I have made more versions, but this is my latest one.
  • Patformer tests. Some tests in making a platformer rpg.
  • Battilde Server Client. A multiplayer ascii graphics monster shooting game.
  • Asciifarm Server Client. A multiplayer ascii graphics rpg/farming game.
  • Contributions to Unknown Horizons. An open-source game based on the Anno series. My contributions include a properly rotated minimap and many bug fixes.
  • Invected: Survival of the Sickest. Project for the Global Game Jam 2018.
  • Asteroids. Made in collaboration with Qqwy.
  • Achtung, die Kurve. One of my oldest projects. Made in Game Maker. Built for windows, but should work fine within wine.
  • Tron. A multiplayer ascii graphics tron game
